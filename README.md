# Avature ATS Scraper

A scraper to get job posts from Avature websites.

## How to Install

```bash
pip install -r requirements.txt
```

## How to Use

```bash
python main.py
```

## Project Structure

```
avature-scraper/
├── input/              # Input files (URLs, sites)
├── output/             # Jobs and stats
├── scripts/            # Helper scripts
├── src/                # Main code
├── main.py             # Start here
├── requirements.txt    # Dependencies
└── README.md           # This file
```
